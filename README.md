# One Toolbox

A fast, offline-capable developer toolbox built with React and Material UI.

**Live site:** https://yanym.github.io/One-Toolbox

## Tools

| Tool | Description |
|---|---|
| JSON Formatter | Validate, format, and minify JSON |
| JSON Diff | Visual side-by-side comparison of two JSON objects |
| JSON Viewer | Explore JSON as an interactive tree |
| JSON Serializer | Convert JSON to code in multiple languages |
| String Escape | Escape/unescape strings (JSON, HTML, URL, regex, SQL, …) |
| XML Formatter | Format and validate XML documents |
| Base64 Converter | Encode and decode Base64 strings |
| Protobuf Converter | Encode and decode Protocol Buffers |
| URL Encoder/Decoder | Encode/decode URLs with nested query parameter parsing |
| Timestamp Converter | Convert Unix timestamps (s/ms/µs/ns), ISO 8601, and date strings across all formats and US timezones |

## Running locally

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000).

## Deploying

Pushes to `main` automatically deploy to GitHub Pages via the included workflow.
